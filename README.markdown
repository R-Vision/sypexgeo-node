# sypexgeo

Sypex Geo API for Node.JS

## Getting Started
```javascript
var sypex = require('sypexgeo');
var geo = new sypex.Geo('path/to/SypexGeoCity.dat', sypex.MODE_MEMORY);
var cityInfo = geo.getCityFull('ip here');
```
